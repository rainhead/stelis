# docs/

Working notes, research, and design detail for Stelis.

The primary orienting documents live at the repo root, not here:

- [`DESIGN.md`](../DESIGN.md) — what Stelis is and the settled commitments.
- [`ROADMAP.md`](../ROADMAP.md) — planning horizons (we are in Horizon 0).
- [`CLAUDE.md`](../CLAUDE.md) — operating instructions for agents.

This directory holds the material that accretes *around* those: research
findings, deeper design notes, and investigations. It is indexed by
[qmd](https://github.com/tobil/qmd) as the `stelis-docs` collection (root `.md`
files are `stelis-root`), so anything dropped here becomes searchable.
